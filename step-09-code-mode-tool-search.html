조립 10단계 · prepStages · 조건부

code-mode · tool-search

도구가 많을 때 모델에 어떻게 노출할지(카탈로그)를 적용하는 단계. 단, 조건부로만 발화한다.

무엇을 하나

  • 도구 노출 카탈로그를 적용 — applyCodeModeCatalog / applyToolSearchCatalog / applyToolSchemaDirectoryCatalog.
  • 한 호출 지점에서 code-mode 또는 tool-search 중 하나의 이름으로 마크된다.

코드 위치 (검증됨)

  • attempt.ts:1761 prepStages.mark (카탈로그 적용 지점)
  • attempt.ts:1713–1745 카탈로그 적용 함수들

왜 "보통 9개"인가

이 5번째 마크는 if (toolSearch.compacted && !toolSearch.catalogReused) 안에서만 발화한다(attempt.ts:1761). 일반적인(비압축) 실행에선 생략되어 추적기에 9개 단계만 남는다 — "10단계"는 이 단계까지 포함한 최대치다.
